Figure 4. The subspace representation  gamma W acts as a linear probe for W . The histograms show  gamma T W lambda ( x fr j ) vs.  gamma T W lambda ( x es j ) (left) and  gamma T Z lambda ( x fr j ) vs.  gamma T Z lambda ( x es j ) (right) for W = French => Spanish and Z = male => female , where { x fr j } and { x es j } are random contexts from French and Spanish Wikipedia, respectively. We also see that  gamma Z does not act as a linear probe for W , as expected.
